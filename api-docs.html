Ghost Nav API v1

Privacy-first routing engine that calculates camera-avoidance routes and surveillance density scores. All endpoints return JSON. CORS is open — call from any origin.

Base URL:
POST /route GET /score ← Back to App
POST /api/v1/route

Compute a privacy-optimized "ghost" route between two coordinates. Returns both the fastest route and a camera-avoidance route, including geometry, distance, duration, camera counts, and a privacy score. The ghost algorithm uses OSRM alternatives + geometric cluster bypasses to minimize ALPR camera exposure.

Request Body (JSON)

Parameter Type Required Description
start [lat, lon] required Origin as [latitude, longitude] array
end [lat, lon] required Destination as [latitude, longitude] array
mode string optional fastest | ghost | both (default: both)

⚡ Try It

Pre-filled: Summerville, SC → Charleston International Airport

Response

        

      


      
      

        Code Examples
        

          
          
          
        

        

          
curl -X POST \
  "http://localhost:8766/api/v1/route" \
  -H "Content-Type: application/json" \
  -d '{
    "start": [33.0185, -80.1786],
    "end":   [32.8986, -80.0407],
    "mode":  "both"
  }'

        

        

          
import requests

resp = requests.post(
    "http://localhost:8766/api/v1/route",
    json={
        "start": [33.0185, -80.1786],
        "end":   [32.8986, -80.0407],
        "mode":  "both"
    }
)
data = resp.json()
print(f"Privacy score: {data['ghost']['privacy_score']}")
print(f"Cameras avoided: {data['cameras_avoided']}")

        

        

          
const resp = await fetch('/api/v1/route', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    start: [33.0185, -80.1786],
    end:   [32.8986, -80.0407],
    mode:  'both'
  })
});
const data = await resp.json();
console.log('Privacy score:', data.ghost.privacy_score);
console.log('Cameras avoided:', data.cameras_avoided);

        

      

    

  


  
  

    

      GET
      /api/v1/score
    

    

      

        Get the surveillance density and privacy score for a geographic area. Returns the number of ALPR
        cameras within the specified radius, camera density per km², and a 0–100 privacy score.
        Useful for assessing a neighbourhood's surveillance exposure before routing.

            Parameter
            Type
            Required
            Description
          

        
        
          

            lat
            float
            required
            Center latitude of search area
          

          

            lon
            float
            required
            Center longitude of search area
          

          

            radius_m
            float
            optional
            Search radius in metres. Default: 1000

import requests

resp = requests.get(
    "http://localhost:8766/api/v1/score",

)

print(f"Privacy score: {data['privacy_score']}/100")
print(f"Cameras nearby: {data['camera_count']}")
print(f"Density: {data['surveillance_density']} cameras/km²")

        

        

          
const params = new URLSearchParams({
  lat: 33.0185, lon: -80.1786, radius_m: 1000
});
const resp = await fetch(`/api/v1/score?${params}`);
const data = await resp.json();
console.log(`Privacy score: ${data.privacy_score}/100`);
console.log(`Cameras: ${data.camera_count}`);
